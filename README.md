### alpine-libv8
```sh
docker build --build-arg V8_VERSION=6.7.31 --build-arg V8_DIR=/usr/local/v8 .
```
#### docker-compose
```yaml
build:
  context: ./alpine-libv8/6.7
  args:
    - V8_VERSION=6.7.31
    - V8_DIR=/usr/local/v8
```
#### Extract
```sh
docker run -v $(pwd):/archive alpine_libv8 sh -c 'tar cvzf /archive/libv8-${V8_VERSION}.tar.gz -C ${V8_DIR} .'
```
